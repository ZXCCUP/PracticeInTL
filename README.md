# TravelLine Tech Practice Project

Мини-копия главной страницы travelline.tech для учебной практики.

## День 3

Сделано:
- создана структура проекта;
- подключены статические файлы из public;
- сверстаны первые блоки главной страницы:
  - Hero;
  - О TravelLine / статистика;
  - Команда;
  - Развиваем платформу;
  - Логотипы клиентов;
- добавлены анимации и интерактивность:
  - изменение цвета шапки при скролле;
  - слайдер команды;
  - открытие информации по клику на элементы платформы;
  - бегущая строка логотипов.

## День 4

Сделано:
- спроектирована структура данных в формате JSON;
- создан файл data/data.json;
- создан сервер на Express;
- сервер отдаёт статическую главную страницу;
- сервер раздаёт папку public со стилями, скриптами и изображениями;
- добавлен API-роут /api/data для чтения данных из JSON;
- проверено открытие страницы через http://localhost:3000.

## День 5

Сделано:
- создана страница admin.html;
- разработана базовая админ-панель;
- добавлены формы для редактирования основных блоков сайта;
- реализована загрузка данных через API `/api/data`;
- добавлен POST-запрос для сохранения изменений;
- настроено сохранение данных в `data/data.json`;
- проверена работа админ-панели и сохранение изменений.

## День 6

Сделано:
- подключена публичная страница к данным из `data/data.json`;
- создан файл `render-page.js` для отображения данных на главной странице;
- реализовано обновление блоков Hero, преимуществ, команды, платформы и клиентов;
- проверен цикл: изменение данных в админке → сохранение в JSON → отображение на главной странице;
- вынесены функции чтения и записи JSON в отдельный модуль;
- добавлены первые автоматические тесты для логики добавления, изменения, удаления и сохранения данных.
